# PicturePerfect API
Deze repository bevat de backend van mijn applicatie voor web-IV.

### Extra technologie
De mogelijkheid om afbeeldingen te uploaden en weer te geven.

### How to
De applicatie en api staan niet online, deze wensen dus offline gerunt te worden.
Als deze wordt opgestart zal deze runnen op localhost:5001. En zal swagger te vinden zijn op localhost:5001/swagger/index.html.

#### Testen
Voor de authenticatie te testen in swagger moet de definitie rechtsboven veranderd worden naar apidocs.
